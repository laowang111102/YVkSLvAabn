# 学生档案管理系统 python645

## 项目概述

学生档案管理系统是一款集学生档案管理、成绩录入、品德评价、社会实践记录及综合评价生成等功能为一体的应用系统。该系统致力于一站式解决院校素质评价需求，采用分层架构设计，代码规范，易于进行二次开发。

## 技术栈

- **后端语言**: Python
- **框架**: Flask
- **数据库**: MySQL

## 功能模块

### 学生端

- 学生档案管理
- 成绩查询
- 品德评价查询
- 社会实践记录查询
- 综合评价查看

### 教师端

- 成绩录入与管理
- 品德评价
- 社会实践记录管理
- 学生综合评价管理

### 管理员端

- 用户权限管理
- 数据维护与备份
- 系统设置

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- **操作系统**: Windows 7/8/10/XP

## 目录结构

```plaintext

├── app                # Flask 应用目录

```

## 核心亮点

- **安全性**: 支持用户权限管理、数据加密、HTTPS传输，确保信息安全。
- **可扩展性**: 分层架构设计，代码规范，便于系统扩展与二次开发。
- **兼容性**: 适配多种操作系统，开箱即用。
- **文档完备**: 提供完整设计报告、数据库表结构、接口文档，助力快速上手与开发。
